Este repositorio contiene una implementación práctica de un Sistema de Control de Estudiantes y Calificaciones, desarrollado tanto en C++ como en Java. El software permite 
registrar estudiantes, calcular automáticamente sus promedios académicos, determinar su estado (Aprobado/Reprobado) y generar reportes filtrados. Como lo solicito el docente, con 
el fin de que nosotros como estudiantes aprendamos a manejar este tipo de programas de la forma correcta, aplicando lo aprendido en clases como en este caso son los diferents tipos 
de Arreglos, Constructores y Encapsulamiento.
